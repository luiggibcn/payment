## Split a table

Useful when a large table is physically separated into smaller ones.

1. Tap **one table** to select it
2. Tap the **âŠ— Split** button in the top toolbar
3. Enter how many tables you want to split it into
4. The panel shows a **preview** with the table number and seats
   assigned to each resulting table
5. Tap **Split** to confirm

Seats are distributed evenly. If the number does not divide equally,
the first tables receive one extra seat.

> You can split any table, not only ones that have been previously
> merged. For example: a 6-seat table that is physically three
> 2-seat tables.

## Add a new table

Tap the **+ New table** button in the top toolbar.

You can configure:
- Table **number**
- **Seats** (minimum 2)
- **Visual size**: Normal (occupies 1 cell) or Large (occupies 2 cells)

The new table is placed automatically in the first available cell of
the active zone.
